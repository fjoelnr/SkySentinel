A versatile and modular weather station using ESP32 microcontrollers, with support for multiple sensors and data transmission options.

This project is designed to be easily extended and customized to suit your needs. Whether you want to use different sensors or transmit data to different platforms, the Modular Weather Station for ESP32 makes it simple to create your own personalized weather station.\hypertarget{md__c__work_projects__sky_sentinel__r_e_a_d_m_e_autotoc_md20}{}\doxysubsection{Features}\label{md__c__work_projects__sky_sentinel__r_e_a_d_m_e_autotoc_md20}

\begin{DoxyItemize}
\item Support for BME280 sensor (temperature, humidity, and pressure)
\item Wi-\/\+Fi connectivity
\item MQTT data transmission
\item Easy integration with other sensors and data platforms
\item Compatible with ESP32-\/\+S2-\/\+Saola-\/1 and ESP32-\/\+S2-\/\+Kaluga-\/1 development boards
\end{DoxyItemize}

Eine vielseitige und modulare Wetterstation mit ESP32-\/\+Mikrocontrollern, die Unterstützung für mehrere Sensoren und Datenübertragungsoptionen bietet.

Dieses Projekt wurde entwickelt, um einfach erweitert und an Ihre Bedürfnisse angepasst zu werden. Ob Sie verschiedene Sensoren verwenden oder Daten auf unterschiedliche Plattformen übertragen möchten, die modulare Wetterstation für ESP32 macht es einfach, Ihre eigene personalisierte Wetterstation zu erstellen.\hypertarget{md__c__work_projects__sky_sentinel__r_e_a_d_m_e_autotoc_md28}{}\doxysubsection{Funktionen}\label{md__c__work_projects__sky_sentinel__r_e_a_d_m_e_autotoc_md28}

\item Unterstützung für BME280-\/\+Sensor (Temperatur, Luftfeuchtigkeit und Druck)
\item WLAN-\/\+Konnektivität
\item MQTT-\/\+Datenübertragung
\item Einfache Integration mit anderen Sensoren und Datenplattformen
\item Kompatibel mit ESP32-\/\+S2-\/\+Saola-\/1 und ESP32-\/\+S2-\/\+Kaluga-\/1-\/\+Entwicklungsboards

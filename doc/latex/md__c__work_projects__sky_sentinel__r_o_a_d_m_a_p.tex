This roadmap outlines the planned features and improvements for the Modular Weather Station project.


\begin{DoxyEnumerate}
\item {\bfseries{Integrate additional sensors\+:}}
\begin{DoxyItemize}
\item Anemometer (Wind speed sensor)
\item Wind vane (Wind direction sensor)
\item Rain gauge (Precipitation sensor)
\item UV sensor
\item Soil moisture sensor
\item Lightning sensor
\end{DoxyItemize}
\item {\bfseries{Web application development\+:}}
\begin{DoxyItemize}
\item User interface for displaying real-\/time weather data
\item Historical data visualization and analysis
\item Configuration and management of weather station settings
\end{DoxyItemize}
\item {\bfseries{Web application on the ESP32\+:}}
\begin{DoxyItemize}
\item Minimalist web application with live data and configuration options hosted on the ESP32
\end{DoxyItemize}
\item {\bfseries{API development\+:}}
\begin{DoxyItemize}
\item Develop an API to expose weather data and allow configuration updates
\end{DoxyItemize}
\item {\bfseries{Server-\/based web application (example for Raspberry Pi)\+:}}
\begin{DoxyItemize}
\item General server-\/based web application, exemplarily set up for Raspberry Pi a. Set up the server environment (consider using Docker for all components, including Influx\+DB, Grafana, etc.) b. Create a web frontend for displaying data and managing the weather station c. Integrate Influx\+DB for storing and managing weather data d. Integrate Grafana for data visualization and analysis e. Implement an alerting function for specific weather conditions
\end{DoxyItemize}
\item {\bfseries{OTA (Over-\/the-\/\+Air) functionality\+:}}
\begin{DoxyItemize}
\item Implement OTA updates for the weather station firmware
\end{DoxyItemize}
\item {\bfseries{Integration with home automation systems\+:}} a. Integrate with io\+Broker b. Integrate with Home Assistant
\end{DoxyEnumerate}\hypertarget{md__c__work_projects__sky_sentinel__r_o_a_d_m_a_p_autotoc_md36}{}\doxysubsection{Deutsch}\label{md__c__work_projects__sky_sentinel__r_o_a_d_m_a_p_autotoc_md36}

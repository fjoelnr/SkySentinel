We follow a Git-\/\+Flow-\/inspired branching model to ensure well-\/organized development\+:


\begin{DoxyEnumerate}
\item {\ttfamily main} branch\+: This branch contains the stable version of the code ready for production. Each release is tagged in this branch.
\item {\ttfamily develop} branch\+: This branch serves as the basis for active development and integration of new features. All new developments should take place in feature branches and be merged back into the {\ttfamily develop} branch.
\item Feature branches\+: These branches are created for the development of new features or improvements. They are created from the {\ttfamily develop} branch and merged back into it once development is complete.

Example\+: {\ttfamily feature/temperature-\/sensor}
\item Release branches\+: When the {\ttfamily develop} branch has reached sufficient stability and a new version is to be released, a release branch is created. Final changes and bug fixes are made in this branch before merging the release into the {\ttfamily main} branch.

Example\+: {\ttfamily release/v1.\+0.\+0}
\item Hotfix branches\+: These branches are created when urgent bug fixes are required in the production version. They are created from the {\ttfamily main} branch and merged back into both the {\ttfamily main} and {\ttfamily develop} branches once the bug fix is complete.

Example\+: {\ttfamily hotfix/1.\+0.\+1}
\end{DoxyEnumerate}

Please follow this branching model when contributing to the project.\hypertarget{md__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md11}{}\doxysection{Submitting Pull Requests}\label{md__c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md11}
When you\textquotesingle{}re ready to submit your changes, please create a pull request against the appropriate branch (usually {\ttfamily develop}). Be sure to describe your changes in detail and reference any related issues.

Your pull request will be reviewed by the maintainers, who may request changes or provide feedback. Once your changes have been approved, they will be merged into the appropriate branch.

Thank you for your contributions!

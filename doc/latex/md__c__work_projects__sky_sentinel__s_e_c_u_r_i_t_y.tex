\hypertarget{md__c__work_projects__sky_sentinel__s_e_c_u_r_i_t_y_autotoc_md40}{}\doxysection{Reporting a Vulnerability}\label{md__c__work_projects__sky_sentinel__s_e_c_u_r_i_t_y_autotoc_md40}
If you discover a security vulnerability in the Modular Weather Station for ESP32 project, please do not create a public issue on Git\+Hub. Instead, send an email to the project maintainers at \href{mailto:thrudhamar@web.de}{\texttt{ thrudhamar@web.\+de}} with the following information\+:


\begin{DoxyItemize}
\item A detailed description of the vulnerability
\item Steps to reproduce the vulnerability
\item The potential impact of the vulnerability on users or their data
\item Any suggestions you may have for resolving the vulnerability
\end{DoxyItemize}

We take all vulnerability reports seriously and will investigate each one. If we confirm the vulnerability, we will work on a patch and release it as soon as possible. We will also credit you for discovering and reporting the vulnerability, unless you prefer to remain anonymous.

Please act responsibly when reporting vulnerabilities and refrain from disclosing the issue publicly or exploiting it for personal gain.

Thank you for helping us keep the Modular Weather Station for ESP32 project secure!
